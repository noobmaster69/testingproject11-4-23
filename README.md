# Unit Lab

A modern React + TypeScript conversion workspace with multi-category unit conversion, theme presets, favorites, profiles, natural-language parsing, and GitHub Pages deployment.

## Features

- 12+ converter categories including pressure, energy, data size, and angle.
- Dark, light, and auto themes with visual presets.
- Favorites, recent history, and saved profiles.
- Time tools: decimal hours <-> clock time.
- Precision controls and scientific notation.
- Natural-language input and browser voice input hook.
- Live currency conversion panel.
- PWA-ready build and offline caching setup.

## Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run test` - Run Vitest + TypeScript checks
- `npm run test:watch` - Run tests in watch mode
- `npm run lint` - Run ESLint
- `npm run format` - Run Prettier
